从 1.5.4 升级到 1.6.0

升级之前你需要断开网络，这样才能用静态文件index.php替换你原来的文件.

第一步: 更新你的CodeIgniter文件

在你的"system"目录下，用新版本的文件替换你原来的文件夹和文件:

  • system/codeigniter
  • system/database
  • system/helpers
  • system/libraries
  • system/plugins
  • system/language

Note:如果在这个目录下有你自定义的文件，需要在更新之前做一下备份.

第二步:在你的配置文件config.php中添加time_to_update

使用其他的session配置变量添加如下内容到system/application/config/config.php文件中

$config['sess_time_to_update'] = 300;

第三步: 添加 $autoload['model']

添加如下内容到 system/application/config/autoload.php

/*
| -------------------------------------------------------------------
| Auto-load Model files
| -------------------------------------------------------------------
| Prototype:
|
| $autoload['model'] = array('my_model');
|
*/

$autoload['model'] = array();

第四步: 修改你的数据库配置文件database.php

把下面的变化添加到system/application/config/database.php这个文件中:

在数据库配置选项的最上面添加变量$active_group

$active_record = TRUE;

删除如下数据库配置选项:

$db['default']['active_r'] = TRUE;

在你的配置文件database.php中添加下列内容：

$db['default']['char_set'] = "utf8";
$db['default']['dbcollat'] = "utf8_general_ci";

第五步: 更新你的用户参考手册

直接用新版本的替换旧版本即可.
